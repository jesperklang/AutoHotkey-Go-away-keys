# AutoHotkey
## Included in Script
* Deactivates Caps lock
* Disables the Insert key
* Disables the Copilot key

### Temporary overrides
* Toggle Caps lock with ```Win + Caps lock```
* Toggle Insert with ```Win + Insert```

## What?
This script is made to be used with [AutoHotkey](https://www.autohotkey.com/), which  is a free, open-source custom scripting language for Windows that allows users to create custom keyboard shortcuts and much more.
